Github Commands

  • git status

    describes our git condition

  • git add .

    stage everything that we changed

  • git commit -m "message"

    puts a label on the changes

  • git push

    pushes all work to origin/branch

  • git checkout -b `name of branch`

    creates a new branch with the specified name

  • git checkout `name of branch`

    switches
