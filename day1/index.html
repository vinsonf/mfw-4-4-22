Github Commands

  • git status

    describes our git condition

    Pictured Below: git status after making changes

  • git add .

    Stages everything we changed

    Pictured Below: git status after running git add .

  • git commit -m "message"

    Puts a label on the changes

    Pictured Below: git status after running git commit -m "message"

  • git push

    Pushes all committed changes to remote(github)

    Pictured Below: git status after running git push

  • git checkout -b `name of branch`

    Creates a new branch with the specified name

  • git checkout `name of branch`

    switches to specified branch

Github Flow

  • Create a new branch

    git checkout -b `name of branch`

  • Make changes

    git add .

  • Commit changes

    git commit -m "message"

  • Push changes

    git push
